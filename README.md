# Continuous and Repeated Prisoner's Dilemma

A mathematical investigation of how strategic behaviour changes when the
Prisoner's Dilemma is extended to continuous actions and repeated interaction.

## Overview

This group project studies variants of the Prisoner's Dilemma beyond the
standard one-shot, two-action game. It examines continuous strategy spaces,
repeated play, cooperation, retaliation, and the long-run behaviour of competing
strategies.

The project combines analytical game theory with computational experiments.

## Limitations and Further Work

Results depend on the selected payoff functions, strategy set, time horizon, and
assumptions about information and memory. Further work could study evolutionary
dynamics, noisy actions, larger populations, and reinforcement-learning agents.
